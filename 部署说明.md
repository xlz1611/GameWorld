## 🚀 部署步骤（推荐方法）

### 方法一：通过 Vercel 网站部署（最简单）

1. **访问 Vercel 网站**
   - 打开浏览器访问：https://vercel.com

2. **导入项目**
   - 点击 "Add New" -> "Project"
   - 在 "Import Git Repository" 中找到 `GameWorld` 项目
   - 点击 "Import"

3. **配置项目**
   - Project Name: `gamehub`（或自定义）
   - Framework Preset: `Next.js`（会自动检测）
   - Root Directory: `./`（保持默认）
   - Build Command: `npm run build`（自动填充）
   - Output Directory: `.next`（自动填充）

4. **添加环境变量**（可选）
   在 "Environment Variables" 部分添加：
   ```
   NEXT_PUBLIC_APP_NAME=GameHub
   NEXT_PUBLIC_ENABLE_ANALYTICS=false
   NEXT_PUBLIC_ENABLE_ERROR_REPORTING=true
   NEXT_PUBLIC_IMAGE_QUALITY=80
   CACHE_TTL=300
   CACHE_MAX_SIZE=100
   MAX_FILE_SIZE=104857600
   ALLOWED_FILE_TYPES=.apk,.zip
   ```

5. **部署**
   - 点击 "Deploy" 按钮
   - 等待构建完成（通常 1-3 分钟）

6. **获取访问地址**
   - 部署成功后会显示一个 `.vercel.app` 域名
   - 例如：`https://gamehub-xyz.vercel.app`

### 方法二：通过 Vercel CLI 部署

由于项目包含 node_modules（470MB），直接上传会超过限制。
建议使用方法一，或者先清理 node_modules：

```bash
cd "C:\Users\111\Desktop\游戏网站"
# 备份 node_modules
mv node_modules node_modules_backup
# 部署
vercel --scope 3750398996-qqcoms-projects --prod
# 恢复 node_modules
mv node_modules_backup node_modules
```

## 📝 部署后配置

### 1. 自定义域名（可选）
- 在 Vercel 项目设置中添加自定义域名
- 需要在域名 DNS 中添加 CNAME 记录

### 2. 数据库配置（如果需要）
项目使用了 PostgreSQL，可以配置：
- Vercel Postgres（免费额度 256MB）
- Supabase（免费额度 500MB）
- Neon（免费额度）

### 3. 文件存储（如果需要）
如果需要上传 APK 文件，可以配置：
- Vercel Blob（免费额度 1GB）
- Cloudflare R2（免费额度 10GB）

## 🔧 常见问题

### Q: 部署失败怎么办？
A: 检查 Vercel 的构建日志，通常是因为：
- 依赖安装失败
- 环境变量缺失
- 构建命令错误

### Q: 如何更新网站？
A: 推送代码到 GitHub，Vercel 会自动重新部署：
```bash
git add .
git commit -m "update"
git push
```

### Q: 如何查看部署日志？
A: 在 Vercel 网站的项目页面点击 "Deployments" 查看历史记录
